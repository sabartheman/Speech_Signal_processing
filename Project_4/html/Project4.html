A phase phase vocoder is a tool used to compress or expand an audio wave so that changes in playback speed result in the same pitch output. Meaning that if you playback the audio at a faster rate you should be able to hear the same pitch as the original rather then what normally happens, which is the pitch increases and sounds like a tiny person.
